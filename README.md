## 이 책은 무엇인가요?

이 책은 **Claude Code와 TMUX를 조합해 멀티에이전트 AI 팀을 구성하고, Remote-Control로 어디서든 팀을 지휘하는 방법**을 다룬 실전 가이드입니다. AI 에이전트 하나를 쓰는 것에서 벗어나, 6명의 AI 에이전트가 동시에 협업하는 팀 자동화 환경을 직접 만듭니다.

보통 AI를 이렇게 씁니다.

```
나 → Claude에게 질문 → 답변 → 또 질문 → ...
```

이 책은 이걸 가능하게 합니다.

```
나 (스마트폰으로 원격 지시)
  ↓
팀장 AI (Claude Code) → 기획자 AI, 개발자 AI, 리뷰어 AI에게 동시에 배분
```

**핵심 기술 3가지:**

- **Claude Code** — 터미널에서 파일을 직접 읽고 쓰는 AI 에이전트
- **TMUX** — 화면을 여러 칸으로 나눠 멀티에이전트를 동시에 실행
- **Remote-Control** — 스마트폰에서 내 컴퓨터의 AI 팀을 원격 지휘

**한 줄 요약**: *Claude Code로 6명짜리 AI 개발팀을 내 컴퓨터에 만들고, 어디서든 스마트폰으로 지휘한다.*

## Claude Code 멀티에이전트로 이런 것들이 가능해집니다

**출근길 지하철에서 AI 팀에 일을 시킨다**
스마트폰 Remote-Control로 "결제 모듈 버그 수정해줘"라고 보내면, 사무실에 도착했을 때 AI 에이전트가 이미 코드를 고쳐놓고 있습니다.

**멀티에이전트 6가지 일이 동시에 진행된다**
기획자 AI가 설계 문서를 쓰는 동안, 개발자 AI가 코드를 짜고, 리뷰어 AI가 품질을 검토합니다. AI 팀 자동화로 순서를 기다릴 필요가 없습니다.

**컨텍스트가 끊겨도 AI 팀 작업이 이어진다**
새 세션을 시작해도 AI 에이전트가 이전 작업 맥락을 파일에서 불러옵니다. "아까 어디까지 했더라"가 사라집니다.

**배포까지 AI 에이전트가 챙긴다**
코드 작성 → 테스트 → 리뷰 → PR 생성까지 한 흐름으로 이어집니다. 1인 개발자도 AI 팀 덕분에 사람은 최종 승인만 합니다.

## Claude Code 멀티에이전트, 어떻게 가능한가요?

세 가지 핵심 기술을 조합합니다.

**TMUX — Claude Code 에이전트를 여러 명으로 만든다**
터미널 화면을 6칸으로 나누고, 각 칸에서 Claude Code를 독립적으로 실행합니다. 멀티에이전트 각자가 다른 역할을 맡아 동시에 일합니다.

**CLAUDE.md — AI 에이전트에게 역할을 부여한다**
각 AI 에이전트가 시작할 때 읽는 역할 정의 파일입니다. "너는 개발자야, 설계는 네 역할이 아니야"처럼 에이전트 역할 경계를 명확히 합니다.

**Remote-Control — 스마트폰으로 AI 팀을 원격 제어한다**
내 컴퓨터에서 실행 중인 Claude Code 팀에게 스마트폰으로 연결합니다. 코드와 파일은 절대 클라우드로 나가지 않고, 통신 경로만 인터넷을 씁니다.

## 책 구성 (Claude Code 멀티에이전트 완전 정복 로드맵)

각 장은 이전 장 위에 쌓이는 구조입니다. 2장에서 기반을 만들고, 3~5장에서 AI 팀을 구성하고, 6~8장에서 실전 자동화 수준으로 끌어올립니다.

**1장. 들어가며**
왜 AI를 혼자 쓰면 안 되는지, 멀티에이전트 팀이 왜 필요한지 설명합니다. 전체 아키텍처를 미리 보고 이 책이 어디를 향하는지 파악합니다.

**2장. Ubuntu 환경에서 Claude & TMUX 설치**
WSL2 또는 네이티브 Ubuntu 환경에 Claude Code와 TMUX를 설치합니다. AI 팀을 올릴 기반 환경을 만듭니다.

**3장. Claude Code와 TMUX로 멀티에이전트 구성**
TMUX 화면을 6칸으로 나누고 각 칸에 Claude Code를 실행합니다. CLAUDE.md로 팀원별 역할을 정의하고, 셋업 스크립트로 원클릭 팀 환경을 구성합니다.

**4장. Claude Code Remote-Control 기능 설정 및 사용**
Remote-Control을 활성화해 외부에서 로컬 Claude Code 세션에 접근하는 방법을 설정합니다. 서버 모드, 보안 설정, 세션 이름 관리를 다룹니다.

**5장. 휴대폰 Claude 앱에서 Remote-Control 사용**
스마트폰 Claude 앱으로 PC의 AI 팀에 접속합니다. QR 코드 연결, 도구 승인, 푸시 알림으로 작업 완료를 확인합니다.

**6장. 핵심 도구 설치 및 활용**
gstack, superpowers, gsd, RTK, MCP를 설치하고 활용합니다. 각 도구가 AI 팀 자동화에서 어떤 역할을 하는지 실전 예시로 설명합니다.

**7장. 실전 팀 에이전트 운용**
팀 지시 흐름 설계, Bot Mode 활용, 업무 분담 전략을 다룹니다. gstack + GSD + Superpowers를 조합한 Triple Crown 전략으로 팀 자동화 워크플로우를 완성합니다.

**8장. 고급 운용 기법**
실제 멀티에이전트 운영 사례, GitHub Actions 자동화, 컨텍스트 관리, 멀티에이전트 충돌 방지를 다룹니다. 이 책에서 가장 실전적인 장입니다.

**9장. 운영 및 문제 해결**
Remote-Control 인증 오류, TMUX 세션 복구, 토큰 최적화 심화를 다룹니다. 실제 운영 중 마주치는 문제를 해결합니다.

**10장. 마치며**
AI 에이전트 기술의 발전 방향을 전망하고, 커뮤니티 참여 방법을 안내합니다.
