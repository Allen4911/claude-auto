## 05-1. Claude 모바일 앱 설치 및 계정 연결

## 이 장에서 배울 내용

스마트폰으로 PC의 Claude Code 세션을 제어하려면 먼저 Claude 공식 모바일 앱을 설치하고, 데스크톱과 동일한 계정으로 로그인해야 합니다. 이 장에서는 앱 설치부터 계정 연결까지 단계별로 안내합니다.

> 💡 **왜 "같은 계정"이 중요할까요?** Remote Control은 PC와 휴대폰이 동일한 claude.ai 계정으로 로그인되어 있을 때만 서로를 찾아 연결합니다. 계정이 다르면 휴대폰에서 PC 세션이 보이지 않습니다.

앱을 깔고 데스크톱과 같은 계정으로 로그인하면 세션이 연결됩니다. 관건은 PC와 휴대폰의 계정을 맞추는 것뿐입니다. 이 장은 앱 설치부터 계정 연결까지 순서대로 따라갑니다.

같은 회사 직원만 사무실 문을 열 수 있듯, PC와 휴대폰이 같은 계정이라는 출입카드를 가져야 서로의 세션에 접근할 수 있습니다. 다른 계정은 다른 회사 카드와 같아서, 아무리 올바른 앱을 써도 문이 열리지 않습니다.

<hr>

## 준비 사항

Remote Control 기능을 사용하려면 다음 조건을 충족해야 합니다.

| 항목 | 요구 사항 |
|------|-----------|
| Claude 구독 | Pro, Max, Team, Enterprise 중 하나 |
| 계정 | claude.ai 계정 (API 키 계정 불가) |
| 앱 버전 | 최신 버전의 Claude 모바일 앱 |
| 네트워크 | 인터넷 연결 (Wi-Fi 또는 모바일 데이터) |

> **중요:** Remote Control은 API 키 방식 로그인을 지원하지 않습니다. 반드시 claude.ai 계정으로 로그인해야 합니다.

> 💡 claude.ai 구독 등급은 Free·Pro·Max·Team·Enterprise로 나뉩니다. Remote Control은 유료 구독(Pro 이상)에서만 켜지고, Free 계정으로 로그인하면 앱에 Code 탭 자체가 보이지 않습니다.

> 💡 **API 키 계정과 claude.ai 계정의 차이** API 키는 개발자가 Anthropic 서버에 직접 요청을 보낼 때 쓰는 인증 수단입니다. claude.ai 계정은 웹·앱 로그인에 사용하는 일반 계정입니다. Remote Control은 클라우드 동기화 기반이라 API 키 방식을 지원하지 않습니다. 혼동하기 쉬우니, 로그인 화면에서 Google·Apple·이메일 중 하나로 들어가는 방식이 claude.ai 계정 로그인임을 기억하세요.

<hr>

## iOS 설치 (iPhone / iPad)

**Step 1 — App Store 열기**

iPhone 또는 iPad에서 App Store를 실행합니다.

**Step 2 — "Claude" 검색**

하단 검색 탭을 누르고 검색창에 **Claude**를 입력합니다. Anthropic이 개발한 공식 앱인지 확인하세요. 개발사가 **Anthropic**으로 표시된 앱을 선택합니다.

**Step 3 — 다운로드 및 설치**

**받기** 버튼을 탭합니다. Face ID 또는 Touch ID로 인증하면 다운로드가 시작됩니다.

검색 결과가 여럿이라 헷갈릴 수 있는데, 개발사가 Anthropic인지만 제대로 확인하면 나머지는 기기 인증 후 자동으로 진행됩니다.

> 💡 검색 결과에 비슷한 이름의 앱이 여럿 나올 수 있습니다. 개발사가 **Anthropic**으로 표시된 앱만 공식 앱입니다. 다른 개발사 앱은 설치하지 마세요.

**Step 4 — 설치 완료 확인**

홈 화면에 Claude 아이콘(주황·검정 배경에 클로드 심볼)이 생기면 설치 성공입니다. 아이콘을 탭해서 앱이 정상적으로 열리는지 확인합니다.

<hr>

## Android 설치

**Step 1 — Google Play Store 열기**

홈 화면 또는 앱 서랍에서 Play Store를 실행합니다.

**Step 2 — "Claude" 검색 및 설치**

검색창에 **Claude**를 입력하고, Anthropic의 공식 앱을 찾아 **설치** 버튼을 탭합니다.

> **삼성/갤럭시 사용자:** Galaxy Store에도 Claude 앱이 있지만, 최신 Remote Control 기능을 보장하려면 Google Play Store 버전을 권장합니다.

**Step 3 — 권한 허용**

설치 후 앱을 처음 실행하면 카메라 권한을 요청할 수 있습니다. QR 코드 스캔(5-2장)을 위해 **허용**을 선택합니다. 나중에 설정에서도 변경할 수 있습니다.

<hr>

## 계정 로그인

앱 설치 후 계정을 연결합니다.

**Step 1 — 앱 실행 및 로그인 화면 진입**

Claude 앱을 처음 실행하면 환영 화면이 나타납니다. **Continue** 또는 **로그인** 버튼을 탭합니다.

**Step 2 — 로그인 방식 선택**

Google, Apple 계정 또는 이메일로 로그인할 수 있습니다. **반드시 데스크톱 Claude Code와 동일한 계정**을 선택해야 세션이 연결됩니다.

> 💡 **어떤 로그인 방식을 썼는지 확인하는 법** 데스크톱 claude.ai 웹사이트에서 오른쪽 상단 프로필 아이콘 → Settings → Account 메뉴에서 이메일 주소를 확인합니다. 이 이메일(또는 연결된 Google·Apple 계정)과 완전히 동일한 방식으로 모바일 앱에 로그인해야 합니다.

**Step 3 — 인증 완료**

선택한 방식으로 인증을 완료하면 Claude 채팅 화면이 나타납니다. 상단 또는 하단 메뉴에 **Code** 탭이 보이면 Remote Control 기능에 접근할 준비가 된 것입니다.

> 💡 **Code 탭**은 휴대폰에서 PC의 Claude Code 세션으로 들어가는 입구입니다. 이 탭이 보이지 않으면 구독 등급(Pro 이상)이나 앱 버전을 먼저 확인하세요(아래 문제 해결 참고).

PC와 휴대폰은 같은 claude.ai 계정으로 묶여 있을 때만 서로의 세션을 공유합니다. 계정이 곧 둘을 잇는 열쇠라서, 휴대폰의 Code 탭에 PC 세션이 보인다면 두 기기가 같은 계정으로 묶인 것입니다. 목록이 비어 있다면 대개 로그인한 계정이 서로 다른 탓입니다.

<hr>

## 계정 연결 확인

로그인 후 데스크톱과 같은 계정인지 확인합니다.

1. 앱 우측 상단 프로필 아이콘을 탭합니다.
2. **Settings → Account** 메뉴에서 이메일 주소를 확인합니다.
3. 데스크톱에서 `claude` 명령으로 세션을 열고, Claude와의 대화창 안에서 `/status`를 입력합니다. 현재 로그인된 계정 이메일이 출력되면 모바일 앱의 이메일과 대조합니다.

> **참고:** `/status`는 Claude와 대화하는 세션 안에서만 쓰는 슬래시 명령입니다. 터미널에 `claude /status`처럼 단독으로 실행하는 것이 아니라, 먼저 `claude` 명령으로 대화 세션을 연 뒤 입력창에 `/status`를 입력하는 방식입니다.

### 계정 일치 여부 빠른 확인 표

| 확인 위치 | 확인 방법 | 정상 상태 |
|-----------|-----------|-----------|
| 모바일 앱 | Settings → Account → 이메일 | 데스크톱과 동일한 이메일 |
| 데스크톱 | claude 세션 내 `/status` | 모바일과 동일한 계정 표시 |
| 모바일 앱 | Code 탭 → 세션 목록 | PC 세션이 목록에 표시됨 |

> 💡 **가장 빠른 확인법** 모바일 앱 Code 탭을 열었을 때 PC 세션이 하나라도 보이면 계정 연결이 정상입니다. 목록이 완전히 비어 있다면 계정 불일치가 가장 흔한 원인입니다.

## 문제 해결

**앱에 Code 탭이 보이지 않는 경우**
- Pro 이상 구독 여부를 확인합니다. Free 플랜은 Remote Control을 지원하지 않습니다.
- 앱을 최신 버전으로 업데이트합니다.
- 앱을 완전히 종료 후 재시작합니다(백그라운드에서 올려두기만 한 상태에서는 구독 상태가 갱신되지 않을 수 있습니다).

**로그인 오류가 발생하는 경우**
- claude.ai 웹사이트에서 먼저 로그인이 되는지 확인합니다.
- 앱을 완전히 종료 후 재시작합니다.
- 기기 날짜·시각 설정이 자동(인터넷 동기화)으로 되어 있는지 확인합니다. 시각이 크게 틀리면 인증이 실패할 수 있습니다.

**"Full-scope login required" 메시지**
- API 키로 로그인을 시도한 경우 이 오류가 발생합니다. Google 또는 이메일 방식으로 다시 로그인하세요.

**Code 탭은 보이는데 세션 목록이 비어 있는 경우**
- PC에서 Claude Code가 실행 중이고 Remote Control이 활성화되어 있는지 확인합니다.
- PC와 모바일의 로그인 계정(이메일)이 정확히 일치하는지 다시 확인합니다.
- 모바일 세션 목록 화면을 아래로 당겨 새로고침합니다.
