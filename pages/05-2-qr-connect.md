## 05-2. QR 코드로 세션 연결하기

## 이 장에서 배울 내용

PC에서 Claude Code를 실행한 뒤, QR 코드를 스캔하는 것만으로 모바일 앱과 즉시 연결할 수 있습니다. URL을 복사하거나 세션 ID를 입력할 필요 없이, 카메라 한 번으로 연결이 완료됩니다.

연결은 **① PC에 QR 표시 → ② 휴대폰으로 스캔 → ③ 세션 연결 완료**의 세 걸음으로 끝납니다. URL을 복사하거나 세션 ID를 타이핑하는 중간 과정이 전혀 없다는 점이 핵심입니다 — 카메라로 QR을 한 번 비추는 동작 하나가 그 모든 입력을 대신합니다.

> 💡 **QR 코드 연결의 원리**: QR 코드 안에는 세션 주소(`claude.ai/code?session=...`)가 담겨 있습니다. 카메라로 읽으면 그 주소로 바로 연결되므로, 긴 URL을 손으로 입력할 필요가 없습니다.

> 💡 **비유: 명함의 QR 코드** 명함 뒤에 찍힌 QR 코드를 스캔하면 담당자 홈페이지로 바로 이동하듯, PC 터미널의 QR 코드를 스캔하면 해당 세션으로 바로 들어갑니다. URL이 아무리 길고 복잡해도 카메라 한 번이면 됩니다.

<hr>

## 사전 조건

- PC에서 Claude Code가 실행 중이어야 합니다.
- Remote Control이 활성화된 상태여야 합니다. (활성화 방법은 4장 참고)
- Claude 모바일 앱이 설치되고 로그인되어 있어야 합니다. (5-1장 참고)

> 💡 **Remote Control 활성화 확인 방법** Claude Code 세션 안에서 `/remote-control`을 입력해 보세요. Remote Control이 꺼진 상태라면 켜주는 방법을 안내하고, 이미 켜진 상태라면 현재 세션 URL을 보여줍니다.

<hr>

## PC에서 QR 코드 표시하기

### 서버 모드로 실행 중인 경우

터미널에서 `claude --remote-control` 플래그로 서버 모드를 실행했다면, 키보드의 **스페이스바**를 누르면 QR 코드가 터미널 화면에 나타납니다.

```bash
# 서버 모드 실행
claude --remote-control
```

```
[Remote Control 서버 실행 중]
세션: dev-machine-elegant-fox
연결 대기 중...

Space: QR 코드 표시 | W: spawn 모드 전환
```

스페이스바를 누르면:

```
████████████████████████████
██ ▄▄▄▄▄ █▀ █▀▀▀▄▀█ ▄▄▄▄▄ ██
██ █   █ █▄▀▄█▀ █▄█ █   █ ██
...
claude.ai/code?session=xxxx
────────────────────────────
모바일 앱으로 스캔하세요
```

위처럼 터미널에 QR 코드가 ASCII 그림으로 그려지고, 그 아래에 세션 주소와 "스캔하세요" 안내가 함께 표시됩니다. 혹시 화면을 넘겨 QR이 사라졌더라도 당황할 필요 없습니다 — 서버 모드에서는 스페이스바를 한 번 누르면 같은 QR이 언제든 다시 떠오릅니다.

> 💡 **서버 모드란?** `claude --remote-control`로 실행하면 여러 모바일 연결을 받는 "대기 서버"가 됩니다. 이 상태에서 스페이스바를 누르면 언제든 QR 코드를 다시 띄울 수 있습니다.

> 💡 **ASCII QR 코드란?** 터미널은 이미지를 직접 표시할 수 없어서, █(전체 채움)·공백(빈 칸) 같은 특수 문자를 격자로 배치해 QR 패턴을 흉내 냅니다. 카메라 앱은 이 패턴을 일반 QR 코드와 동일하게 읽습니다. 폰트가 깨지거나 터미널이 너무 좁으면 인식이 안 될 수 있습니다 — 이럴 때는 폰트 크기를 키우거나 터미널 창을 넓게 조정하세요.

### 세션 이름 지정하여 서버 모드 실행

QR 스캔 후 모바일에서 어느 세션인지 구분하기 쉽게, 이름을 붙여 실행할 수 있습니다.

```bash
# 이름을 붙여 서버 모드 실행 (이름은 --remote-control 뒤 위치 인수로 전달)
claude --remote-control "민준-아키텍트"
```

이렇게 하면 모바일 세션 목록에 `dev-machine-elegant-fox` 같은 임의 이름 대신 `민준-아키텍트`가 표시됩니다.

### 일반 세션에서 Remote Control을 켠 경우

`/remote-control` 명령으로 현재 세션을 원격 제어 가능 상태로 만들면, 세션 URL이 표시됩니다. 이 URL을 모바일 브라우저로 직접 열 수도 있지만, QR 코드를 원한다면 다음 방법을 사용합니다.

터미널에서 별도의 서버 모드를 띄운 뒤 스페이스바로 QR 코드를 표시하는 것이 가장 간편합니다.

## 모바일 앱에서 QR 코드 스캔하기

### iOS (iPhone / iPad)

**Step 1 — Code 탭 이동**

앱 하단 탭 바에서 **Code** 아이콘을 탭합니다.

**Step 2 — QR 스캔 버튼 탭**

Code 화면 우측 상단에 있는 **QR 코드 스캔** 아이콘(카메라 또는 격자 모양)을 탭합니다.

**Step 3 — 카메라로 QR 코드 향하기**

카메라 뷰파인더가 열립니다. PC 터미널에 표시된 QR 코드를 화면 중앙에 맞춥니다. 잠시 후 자동으로 인식됩니다.

> 💡 **인식 팁** QR 코드 전체가 화면 안에 들어와야 합니다. 너무 가까이 대면 QR 일부가 잘리고, 너무 멀리 대면 해상도가 부족합니다. 카메라와 화면 사이 **15~30cm** 거리가 가장 잘 인식됩니다.

**Step 4 — 연결 확인**

인식 즉시 세션 연결 화면으로 전환됩니다. PC에서 실행 중인 Claude Code의 현재 대화 내용이 모바일 화면에 나타나면 연결 성공입니다.

<hr>

### Android

**Step 1 — Code 탭으로 이동**

앱 하단 내비게이션에서 **Code**를 탭합니다.

**Step 2 — 연결 추가**

화면 우측 상단의 **+** 버튼 또는 **세션 연결** 옵션을 탭합니다.

**Step 3 — QR 코드 스캔 선택**

연결 방법 선택 시트에서 **QR 코드 스캔**을 선택합니다.

**Step 4 — 스캔 및 연결**

카메라를 PC 터미널의 QR 코드에 맞추면 자동 인식 후 세션이 열립니다.

> 💡 **카메라 권한이 없을 경우** Android 설정 → 앱 → Claude → 권한 → 카메라를 허용으로 변경한 뒤 다시 시도하세요.

<hr>

## 연결 후 화면 구성

세션 연결에 성공하면 다음 요소가 화면에 나타납니다.

| 구성 요소 | 설명 |
|-----------|------|
| 세션 이름 | 상단 타이틀바 (예: "dev-machine-elegant-fox") |
| 대화 기록 | PC에서 진행 중인 대화 내용 실시간 표시 |
| 입력창 | 하단 텍스트 입력 필드 |
| 상태 표시 | 연결 상태 아이콘 (초록색 = 연결됨) |

정리하면 모바일 세션 화면은 위에서 아래로 **세션 이름(타이틀바) → 대화 기록(가운데) → 입력창(하단)**의 익숙한 채팅 앱 구조이고, 한쪽 구석의 작은 상태 아이콘이 연결 여부를 알려 줍니다. 초록 아이콘이면 PC와 정상 연결된 상태이니, 입력창에 명령을 적어 보내면 PC의 세션이 그대로 받아 실행합니다.

### 연결 직후 해볼 수 있는 첫 번째 테스트

연결 직후 입력창에 아래를 입력해 봅니다.

```
안녕, 너는 누구야?
```

PC 세션이 답변하고, 그 내용이 모바일 화면에 실시간으로 나타나면 양방향 연결이 완벽하게 동작하는 것입니다.

## QR 코드가 작동하지 않을 때

**카메라가 인식하지 못하는 경우**
- 터미널 폰트 크기를 키워 QR 코드를 더 크게 표시합니다.
- 방 조명을 밝히거나 모니터 밝기를 높입니다.
- 카메라와 화면 사이 거리를 15~30cm 정도로 조정합니다.
- 터미널 창을 최대화해 QR 코드가 더 많은 픽셀을 차지하도록 합니다.

**"세션을 찾을 수 없음" 오류**
- PC와 모바일이 동일한 claude.ai 계정으로 로그인되어 있는지 확인합니다.
- PC에서 Remote Control이 여전히 실행 중인지 확인합니다 (터미널 창이 닫히지 않았는지).
- QR 코드가 만료됐을 수 있습니다 — 스페이스바를 다시 눌러 새 QR 코드를 띄웁니다.

**스캔은 됐지만 연결이 안 되는 경우**
- 인터넷 연결 상태를 확인합니다.
- PC 방화벽이 443 포트(HTTPS) 아웃바운드를 허용하는지 확인합니다.

**QR 코드가 터미널에 깨져 보이는 경우**
- 터미널 인코딩을 UTF-8로 설정합니다.
- 터미널 폰트를 등폭(monospace) 글꼴로 변경합니다. 비례 폰트는 █ 문자의 너비가 달라져 QR 모양이 무너집니다.
