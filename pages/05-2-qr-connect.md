# 5-2. QR 코드로 세션 연결하기

## 이 장에서 배울 내용

PC에서 Claude Code를 실행한 뒤, QR 코드를 스캔하는 것만으로 모바일 앱과 즉시 연결할 수 있습니다. URL을 복사하거나 세션 ID를 입력할 필요 없이, 카메라 한 번으로 연결이 완료됩니다.

---

## 사전 조건

- PC에서 Claude Code가 실행 중이어야 합니다.
- Remote Control이 활성화된 상태여야 합니다. (활성화 방법은 4장 참고)
- Claude 모바일 앱이 설치되고 로그인되어 있어야 합니다. (5-1장 참고)

---

## PC에서 QR 코드 표시하기

### 서버 모드로 실행 중인 경우

터미널에서 `claude remote-control` 명령으로 서버 모드를 실행했다면, 키보드의 **스페이스바**를 누르면 QR 코드가 터미널 화면에 나타납니다.

```bash
# 서버 모드 실행
claude remote-control
```

```
[Remote Control 서버 실행 중]
세션: dev-machine-elegant-fox
연결 대기 중...

Space: QR 코드 표시 | W: spawn 모드 전환
```

스페이스바를 누르면:

```
████████████████████████████
██ ▄▄▄▄▄ █▀ █▀▀▀▄▀█ ▄▄▄▄▄ ██
██ █   █ █▄▀▄█▀ █▄█ █   █ ██
...
claude.ai/code?session=xxxx
────────────────────────────
모바일 앱으로 스캔하세요
```

`[스크린샷 위치: 터미널에 QR 코드가 표시된 화면]`

### 일반 세션에서 Remote Control을 켠 경우

`/remote-control` 명령으로 현재 세션을 원격 제어 가능 상태로 만들면, 세션 URL이 표시됩니다. 이 URL을 모바일 브라우저로 직접 열 수도 있지만, QR 코드를 원한다면 다음 방법을 사용합니다.

터미널에서 별도의 서버 모드를 띄운 뒤 스페이스바로 QR 코드를 표시하는 것이 가장 간편합니다.

---

## 모바일 앱에서 QR 코드 스캔하기

### iOS (iPhone / iPad)

**Step 1 — Code 탭 이동**

앱 하단 탭 바에서 **Code** 아이콘을 탭합니다.

`[스크린샷 위치: 하단 탭 바 - Code 탭 강조]`

**Step 2 — QR 스캔 버튼 탭**

Code 화면 우측 상단에 있는 **QR 코드 스캔** 아이콘(카메라 또는 격자 모양)을 탭합니다.

`[스크린샷 위치: Code 화면 우측 상단 QR 스캔 아이콘]`

**Step 3 — 카메라로 QR 코드 향하기**

카메라 뷰파인더가 열립니다. PC 터미널에 표시된 QR 코드를 화면 중앙에 맞춥니다. 잠시 후 자동으로 인식됩니다.

`[스크린샷 위치: 카메라 뷰파인더에 QR 코드를 비추는 화면]`

**Step 4 — 연결 확인**

인식 즉시 세션 연결 화면으로 전환됩니다. PC에서 실행 중인 Claude Code의 현재 대화 내용이 모바일 화면에 나타나면 연결 성공입니다.

`[스크린샷 위치: 모바일에서 세션이 열린 화면]`

---

### Android

**Step 1 — Code 탭으로 이동**

앱 하단 내비게이션에서 **Code**를 탭합니다.

**Step 2 — 연결 추가**

화면 우측 상단의 **+** 버튼 또는 **세션 연결** 옵션을 탭합니다.

`[스크린샷 위치: Android Code 화면 상단 + 버튼]`

**Step 3 — QR 코드 스캔 선택**

연결 방법 선택 시트에서 **QR 코드 스캔**을 선택합니다.

`[스크린샷 위치: 연결 방법 선택 - QR 코드 스캔 옵션]`

**Step 4 — 스캔 및 연결**

카메라를 PC 터미널의 QR 코드에 맞추면 자동 인식 후 세션이 열립니다.

---

## 연결 후 화면 구성

세션 연결에 성공하면 다음 요소가 화면에 나타납니다.

| 구성 요소 | 설명 |
|-----------|------|
| 세션 이름 | 상단 타이틀바 (예: "dev-machine-elegant-fox") |
| 대화 기록 | PC에서 진행 중인 대화 내용 실시간 표시 |
| 입력창 | 하단 텍스트 입력 필드 |
| 상태 표시 | 연결 상태 아이콘 (초록색 = 연결됨) |

`[스크린샷 위치: 세션 연결 완료 후 모바일 화면 전체 구성]`

---

## QR 코드가 작동하지 않을 때

**카메라가 인식하지 못하는 경우**
- 터미널 폰트 크기를 키워 QR 코드를 더 크게 표시합니다.
- 방 조명을 밝히거나 모니터 밝기를 높입니다.
- 카메라와 화면 사이 거리를 15~30cm 정도로 조정합니다.

**"세션을 찾을 수 없음" 오류**
- PC와 모바일이 동일한 claude.ai 계정으로 로그인되어 있는지 확인합니다.
- PC에서 Remote Control이 여전히 실행 중인지 확인합니다 (터미널 창이 닫히지 않았는지).

**스캔은 됐지만 연결이 안 되는 경우**
- 인터넷 연결 상태를 확인합니다.
- PC 방화벽이 443 포트(HTTPS) 아웃바운드를 허용하는지 확인합니다.
